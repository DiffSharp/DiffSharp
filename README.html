Running notebooks in MyBinder

The Dockerfile and NuGet.config allow us to run generated notebooks in MyBinder

Since the generated docs are not yet "up", a recent version may be found in the gh-pages branch of dsyme/DiffSharp:

  • gh-pages branch of dsyme/DiffSharp:

  • index.ipynb for dev branch of dsyme/DiffSharp:

This published version of the docs can be refreshed by

  1. Build FSharp.Formatting

2: 
3: 

    git clone https://github.com/fsprojects/FSharp.Formatting  ../tmp/FSharp.Formatting
cd ..
.

  2. Generate Docs (after building)

    rmdir /s /q ..\tmp git clone https://github.com/DiffSharp/DiffSharp ../tmp/DiffSharp-docs -b tmpdocs --depth 1 pushd ..\tmp\DiffSharp-docs git rm -fr * popd ....\DiffSharp ..\FSharp.Formatting\src\FSharp.Formatting.CommandTool\bin\Debug\netcoreapp3.1\fsdocs.exe build --output ..\tmp\DiffSharp-docs pushd ..\tmp\DiffSharp-docs git add . git commit -a -m "commit docs" git push -f https://github.com/DiffSharp/DiffSharp tmpdocs git push -f https://github.com/dsyme/DiffSharp tmpdocs popd ....\DiffSharp
